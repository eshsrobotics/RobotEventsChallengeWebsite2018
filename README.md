# Robotics Website

## How to use
### Prerequisites
* git
* npm (node)
### Setup
```bash
$ git clone https://github.com/eshsrobotics/RobotEventsChallengeWebsite2018
$ cd RobotEventsChallengeWebsite2018
$ npm install
```

### Dev Server
```bash
$ npm run serve
```

### Build for Prod
```bash
$ npm run build
```
